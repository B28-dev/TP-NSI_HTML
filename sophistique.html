1. Quelle structure de données fonctionne selon le principe "premier entré, premier sorti" ?

Pile (stack)
File (queue)
Arbre binaire
Dans une file (queue), le premier élément ajouté est le premier à être retiré (FIFO).
Indice: Pensez à une file d'attente dans la vie réelle.

2. Comment appelle-t-on l'algorithme qui permet de trouver le plus court chemin dans un graphe pondéré ?


L'algorithme de Dijkstra permet de déterminer le plus court chemin entre les nœuds d'un graphe pondéré.
Indice: C'est un algorithme inventé par Edsger W. Dijkstra.

3. Quels sont les formats d'encodage d'images ?

JPEG
MP3
PNG
JPEG et PNG sont des formats d'encodage d'images, tandis que MP3 est un format audio.
Indice: L'un des formats mentionnés est utilisé pour l'audio, pas pour les images.

4. En SQL, quelle commande permet de sélectionner des données dans une table ?

INSERT
SELECT
UPDATE
La commande SELECT permet de récupérer des données d'une ou plusieurs tables.
Indice: Cette commande est suivie de noms de colonnes et du mot-clé FROM.

5. Complétez la fonction Python qui calcule la factorielle d'un nombre :

def factorielle(n):
    if n <= 1:
        return ___
    else:
        return n * factorielle(___)

La factorielle de 1 ou 0 est 1, et la fonction récursive appelle factorielle(n-1).
Indice: Pour la récursion, on doit appeler la fonction avec un argument qui se rapproche du cas de base.

6. Quels sont les principes de la programmation orientée objet ?

Encapsulation
Héritage
Compilation
Polymorphisme
Les principes fondamentaux de la POO sont l'encapsulation, l'héritage et le polymorphisme. La compilation est un processus indépendant.
Indice: Il y a trois principes fondamentaux. L'un des choix proposés concerne la traduction du code.

7. Quelle est la complexité temporelle de l'algorithme de tri à bulles dans le pire des cas ?

O(1)
O(log n)
O(n)
O(n²)
Le tri à bulles a une complexité temporelle de O(n²) dans le pire des cas car il utilise deux boucles imbriquées.
Indice: Pensez aux boucles imbriquées utilisées dans cet algorithme.

8. Dans le modèle TCP/IP, quel protocole est utilisé pour la navigation web ?


HTTP (HyperText Transfer Protocol) est le protocole utilisé pour la navigation web.
Indice: C'est un protocole dont l'acronyme commence par "HT".
